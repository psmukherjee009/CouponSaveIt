Edit the Coupon

If you are the owner of the coupon then you can call the API EDITCOUPON with the pp_coupon_id value and the values to be edited prefixed by "e_".
A html link like this will work. You need to have only the fields which you want to modify.


<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Create PayPal Save Button</title>
	</head>
	<body>
		<form action="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager4.cgi/EDITBUTTON/" method="POST">
<!-- Don't Edit above this line -->

		<input type="hidden" name="pp_coupon_id"		value="PAYPAL_COUPON_ID">
		<input type="hidden" name="e_merchant_name"		value="THE_MERCHANT_NAME">
		<input type="hidden" name="e_merchant_website"	value="THE_MERCHANT_WEBSITE">
		<input type="hidden"	name="e_paypal_merchant_id"	value="PAYPAL_MERCHANT_ID">
		<input type="hidden"	name="e_target_url"		value="COUPON_DESTINATION_URL">
		<input type="hidden"	name="e_image_url"		value="COUPON_IMAGE">
		<input type="hidden"	name="e_coupon_code"		value="COUPON_CODE">
		<input type="hidden"	name="e_expiration_date"	value="EXPIRATION_DATE_OF_COUPON">
		<input type="hidden"	name="e_keywords"		value="KEYWORDS DESCRIBING MERCHANT AND COUPON">

<!-- Don't Edit below this line -->
		<input type="image"	name="choice"	value="couponsave" src="http://www.parthamukherjee.com/images/CouponSaveIt/edit_button.png" >
	</form>
		
	</body>
</html>
All the folks who have saved this coupon on their wallet will not be able to view, edit or use it anymore.
