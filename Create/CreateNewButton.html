Instructions

To Obtain the PayPal coupon save button you need to do the following

  • Cut and paste the following code into a notepad
  • Replace and fill out all the values which are in CAPS. These value should pertain to the coupon you want to save
  • Copy the code with the replaced values in a file and save as html.
  • Invoke the html file in a browser
  • Press the create button


Create PayPal Save Button 

<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Create PayPal Save Button</title>
	</head>
	<body>
		<form action="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager4.cgi/CREATEBUTTON/" method="POST">
<!-- Don't Edit above this line -->

		<input type="hidden"	name="merchant_name"		value="THE_MERCHANT_NAME">
		<input type="hidden"	name="merchant_website"		value="THE_MERCHANT_WEBSITE">
		<input type="hidden"	name="paypal_merchant_id"	value="PAYPAL_MERCHANT_ID">		<!-- if you donâ€™t know the PayPal merchant ID of the merchant, please keep this field empty --!>
		<input type="hidden"	name="target_url"		value="COUPON_DESTINATION_URL">
		<input type="hidden"	name="image_url"		value="COUPON_IMAGE">
		<input type="hidden"	name="coupon_code"		value="COUPON_CODE">
		<input type="hidden"	name="expiration_date"		value="EXPIRATION_DATE_OF_COUPON">
		<input type="hidden"	name="offer_title"		value="TITLE_OF_THE_OFFER">
		<input type="hidden"	name="offer_type"		value="5% off or free shipping">
		<input type="hidden"	name="redemption_point"		value="ONLINE OFFLINE or BOTH">
		<input type="hidden"	name="keywords"			value="KEYWORDS DESCRIBING MERCHANT AND COUPON">

<!-- Don't Edit below this line -->
		<input type="image"	name="choice"	value="couponsave" src="http://www.parthamukherjee.com/images/CouponSaveIt/create_button.gif" >
	</form>
		
	</body>
</html>
