To Obtain the PayPal coupon save button you need to do the following

  • Cut and paste the following code into a notepad
  • Replace and fill out all the values which are in CAPS. These value should pertain to the coupon you want to save
  • Copy the code with the replaced values and place it after the coupon


	<form action="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager.cgi/SAVE/" method="post">
<!-- Don't Edit above this line -->

		<input type="hidden" 		name="merchant_name"		value="MERCHANT NAME">
		<input type="hidden"		name="paypal_merchant_id"	value="PAYPAL MERCHANT ID">
		<input type="hidden"		name="target_url"		value="COUPON DESTINATION URL">
		<input type="hidden"		name="image_url"		value="COUPON IMAGE">
		<input type="hidden"		name="coupon_code"		value="COUPON CODE">
		<input type="hidden"		name="expiration_date"		value="EXPIRATION DATE OF COUPON in (YYYY-MM-DD) FORMAT">
		<input type="hidden"		name="offer_title"		value="EXPLAIN THE COUPON">
		<input type="hidden"		name="offer_type"		value="% OFF, $ OFF, FREE SHIPPING ETC">
		<input type="hidden"		name="redemption_point"		value="ONLINE OR OFFLINE OR BOTH">
		<input type="hidden"		name="keywords"			value="KEYWORDS DESCRIBING MERCHANT AND COUPON">

<!-- Don't Edit below this line -->
		<input type="image"	name="choice"	value="couponsave" src="http://www.parthamukherjee.com/images/CouponSaveIt/save_btn_REG.gif" >
	</form>
