To Obtain the PayPal coupon save button you need to do the following

  • Cut and paste the following code into a notepad
  • Replace and fill out all the values which are in CAPS. These value should pertain to the coupon you want to save
  • Copy the code with the replaced values in a file and save as html.
  • Invoke the html file in a browser
  • Press the create button


Create PayPal Save Button 

<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Create PayPal Save Button</title>
	</head>
	<body>
		<form action="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager4.cgi/CREATEBUTTON/" method="POST">
<!-- Don't Edit above this line -->

		<input type="hidden"	name="merchant_name"		value="THE_MERCHANT_NAME">
		<input type="hidden"	name="merchant_website"		value="THE_MERCHANT_WEBSITE">
		<input type="hidden"	name="paypal_merchant_id"	value="PAYPAL_MERCHANT_ID">		<!-- if you donâ€™t know the PayPal merchant ID of the merchant, please keep this field empty --!>
		<input type="hidden"	name="target_url"		value="COUPON_DESTINATION_URL">
		<input type="hidden"	name="image_url"		value="COUPON_IMAGE">
		<input type="hidden"	name="coupon_code"		value="COUPON_CODE">
		<input type="hidden"	name="expiration_date"		value="EXPIRATION_DATE_OF_COUPON">
		<input type="hidden"	name="offer_title"		value="TITLE_OF_THE_OFFER">
		<input type="hidden"	name="offer_type"		value="5% off or free shipping">
		<input type="hidden"	name="redemption_point"		value="ONLINE OFFLINE or BOTH">
		<input type="hidden"	name="keywords"			value="KEYWORDS DESCRIBING MERCHANT AND COUPON">

<!-- Don't Edit below this line -->
		<input type="image"	name="choice"	value="couponsave" src="http://www.parthamukherjee.com/images/CouponSaveIt/create_button.gif" >
	</form>
		
	</body>
</html>
Click to Use the Create Button Form to create PayPal Save button

View the Coupon

If you are the owner of the coupon then you can call the API VIEWCOUPON with the pp_coupon_id value.
The link can be pasted on browser url with pp_coupon_id filled.


	http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager4.cgi/VIEWCOUPON/?pp_coupon_id=PAYPAL_COUPON_ID
Click to Use the View Button Form to view PayPal Save button contents

Edit the Coupon

If you are the owner of the coupon then you can call the API EDITCOUPON with the pp_coupon_id value and the values to be edited prefixed by "e_".
A html link like this will work. You need to have only the fields which you want to modify.


<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Create PayPal Save Button</title>
	</head>
	<body>
		<form action="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager4.cgi/EDITBUTTON/" method="POST">
<!-- Don't Edit above this line -->

		<input type="hidden" name="pp_coupon_id"		value="PAYPAL_COUPON_ID">
		<input type="hidden" name="e_merchant_name"		value="THE_MERCHANT_NAME">
		<input type="hidden" name="e_merchant_website"	value="THE_MERCHANT_WEBSITE">
		<input type="hidden"	name="e_paypal_merchant_id"	value="PAYPAL_MERCHANT_ID">
		<input type="hidden"	name="e_target_url"		value="COUPON_DESTINATION_URL">
		<input type="hidden"	name="e_image_url"		value="COUPON_IMAGE">
		<input type="hidden"	name="e_coupon_code"		value="COUPON_CODE">
		<input type="hidden"	name="e_expiration_date"	value="EXPIRATION_DATE_OF_COUPON">
		<input type="hidden"	name="e_keywords"		value="KEYWORDS DESCRIBING MERCHANT AND COUPON">

<!-- Don't Edit below this line -->
		<input type="image"	name="choice"	value="couponsave" src="http://www.parthamukherjee.com/images/CouponSaveIt/edit_button.png" >
	</form>
		
	</body>
</html>
All the folks who have saved this coupon on their wallet will not be able to view, edit or use it anymore.

Click to Use the Edit Button Form to edit PayPal Save button

Delete the Coupon from PayPal

If you are the owner of the coupon then you can call the API DELETEBUTTON with the pp_coupon_id value to delete it. A html link like this will work 


	<a href="http://www.parthamukherjee.com/cgi-perl/GlobalCouponManager.cgi/DELETEBUTTON?pp_coupon_id=PAYPAL_COUPON_ID">
All the folks who have saved this coupon on their wallet will not be able to view, edit or use it anymore.

Click to Use the Delete Button Form to delete PayPal Save button
