Demo: Modified Express Checkout With Coupon Redemption

How To Read the Demo
1. The SetEC button calls the coupon server to get non-expired coupons, and pass them to the merchant cart (for this demo we did not distinguish merchant to save time)
2. The merchant cart applies coupons and updates total (simulated in the js pop up)
3. Merchant cart passes the new $total to the coupon app, and we pass the new amount to EC: you see the success message
